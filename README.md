## 🙋🏻 소개
> 개발 기간: 24.02 ~ 진행중

소셜 엔터테인먼트 서비스의 일종으로, 유저들의 소통과 트렌드 파악을 돕는 스토리텔링형 밸런스게임 플랫폼 서비스예요.    
유저들이 직접 자신의 고민을 투표와 함께 업로드하는 커뮤니티 타입의 ‘톡픽’과 가볍게 즐길 수 있는 게임 타입의 ‘밸런스게임’ 으로 구성되어 있어요.
